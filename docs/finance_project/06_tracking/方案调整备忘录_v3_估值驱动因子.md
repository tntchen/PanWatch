# 方案调整备忘录 v3 — 估值驱动因子扩展（2026-07-20）

> 触发：用户指出东芯市值支撑与存储关联有限，更多来自砺算科技与 H 股 IPO 预期。
> 结论：判断属实，且本次暴跌后估值结构已翻转。方案升级为"存储基本面 + 三大非存储因子"双轨框架。

## 一、估值构成拆解：市场到底在给什么定价

### 1. 历史结构（2026 年 1 月口径，市场分析一致）
- 东芯自身存储业务合理估值 50~80 亿（彼时亏损、利基 DRAM 体量小），而市值 541 亿——**85%~90% 的市值由砺算预期驱动**
- 自 2024 年 8 月投资砺算以来，东芯股价最高涨约 8 倍，本质是"买砺算的二级映射"
- 7/1 峰值市值约 926 亿（209.55 元）：按存储 2026E 6.12 亿净利 × 板块中枢 55-60x ≈ 340-370 亿计，**峰值时市场给"砺算+WiFi7+H股"打包定价约 550 亿**

### 2. 当前结构（7/20 收盘 100.38 元，市值 444 亿）
- 存储主业按 2026E 一致预期 6.12 亿 × 可比公司现价 PE（兆易 72x/普冉 73x，均已同步下跌）≈ **440 亿**
- 即：**存储基本面以同业平价即可解释当前全部市值，砺算 + WiFi7 + H 股三重期权的市场定价已压缩至 ≈0**
- 这正是与报告时点（7/16，99x PE、溢价 36-39%）的本质区别——估值支撑的主力已从"预期"切换回"基本面"

### 3. 推断
- 若存储涨价逻辑兑现（8/22 半年报验证），当前价位下行有基本面托底
- 若砺算/H 股出现正向催化，全部是增量，属"免费期权行权"
- 反之若存储证伪，价格将向纯存储价值回归（基准情景估值锚 76-106 元区间下沿，见原报告第 5 章）

### 因子 B：东芯 H 股发行
- 5/22 董事会、6/8 股东会通过；7/14 投资者关系活动确认"正积极推进"，尚需证监会/联交所审批
- 双向影响：短期摊薄（≤10% 股本 +15% 超配）+ A/H 折价惯例（H 股定价通常较 A 股折让 20-30%）→ 发行窗口临近时对 A 股有压制；长期引入国际流动性、为砺算资本运作提供港币平台

### 因子 C：国产 GPU/算力板块情绪
- 摩尔线程、沐曦已上市，板块情绪高涨期东芯享受高 Beta 映射；7 月全球算力股回调同步放大了东芯跌幅
- 长鑫 7/27 挂牌：既是流动性抽血事件，也可能成为国产半导体情绪修复的观察点

## 三、方案 v3 修订点

### 1. 催化剂日历（在 v2 基础上新增三项）
| 新增监控 | 信号性质 | 动作 |
|---|---|---|
| 砺算 B 轮融资公告 / 大基金进驻 | 强多（续命+背书） | 右侧批触发条件之一可放宽（资金信号豁免） |
| 砺算 IPO 辅导备案（证监会上海局） | 强多（资本路径落地） | 上行情景加码，回看估值上限 |

| 东芯 H 股聆讯/定价公告 | 短空中长多 | 定价公告日前不加仓（折价压制），发行完成后利空出尽可视作右侧信号之一 |

### 2. 逻辑止损条款补充（双轨变三轨）
原两轨：①合约价走平/下跌 ②半年报毛利率<45%。新增：
③**砺算证伪轨**：若出现砺算资金链断裂、核心团队流失、产品召回等实质性恶化——由于当前定价中期权≈0，不触发止损，但**取消"上行情景"的加码动作**，回归纯存储持有逻辑。

### 3. 情景树修订（上行情景正式纳入非存储驱动）
- **上行**：存储兑现（基本盘）+ 任一非存储催化（砺算 B 轮/辅导、H 股落地、GPU 板块情绪修复）→ 修复目标 120-130 不变，若催化强度高可看至 140+
- **基准**：存储兑现 + 期权沉默 → 持有 + 波段，估值锚 100-120
- **下行**：存储证伪 → 回归纯存储价值区间下沿（76-90），执行减仓纪律不变

### 4. 不变的部分
- 四批建仓价位、100 元防线规则、仓位上限、8/22 半年报验证窗口——全部维持 v2

## 四、方法论记录（写入复盘）
- 对"主业+重大股权投资"的双主业映射型标的，估值拆解（SOTP）必须作为标准维度：分别计算"主业可解释市值"与"期权定价"，判断当前价格由哪一端支撑——这决定了止损逻辑该挂在哪个变量上
- 市场情绪期，期权端可贡献 85-90% 市值；恐慌期可压缩至 0——期权定价的波动远大于主业，是这类标的超涨超跌的根源
