# 东芯股份（688110.SH）抄底分析报告：存储超级周期下的滞涨弹性标的

> 报告日期：2026-07-17 ｜ 基准价格：2026-07-16 收盘 136.50 元 ｜ 总市值约 604 亿元
> 数据来源：iFinD、天眼查、TrendForce/集邦、财新、公司公告（详见各章引用）

## Executive Summary（约600字，Round 2 撰写）
### 核心结论
#### 东芯是本轮利基存储涨价周期中"业绩弹性已兑现、股价严重滞涨"的错配标的：SLC NAND 涨价最猛（2026H1 +130~150%），公司 65% 营收来自 NAND，2026Q1 营收 +237%、毛利率 53.17%、净利 1.38 亿扭亏，但 YTD 仅 +3.9%（vs 普冉 +313%、兆易 +140%）
#### 7 月回调系全球存储板块共振 + 主力获利了结（7/1-7/16 净流出约 28 亿），非基本面恶化；但筹码结构受损（股东户数 +140%），抄底须分批 + 事件驱动
#### 估值：2026E 动态 PE 约 99 倍，高于板块中枢 55-60 倍——安全边际不来自估值，而来自涨价持续与一致预期上修；抄底本质是"买涨价持续至 2027 的期权 + 三重免费期权（砺算 GPU/Wi-Fi 7/车规）"
#### 方案要点：分四批建仓（120±3 / 110±3 / 100±3 / 情绪极端位），围绕 8/22 半年报与 10/13 解禁两个日历锚点调整节奏，止损纪律与情景树明确

#### 8.1.1 定位：左侧分批建仓，博取"涨价持续 + 业绩上修 + 滞涨补涨"三重逻辑；非一次性抄底

### 8.2 分批建仓计划
#### 8.2.1 四批触发位：第一批 120±3 元（6 月平台下沿 + 回补缺口区）；第二批 110±3 元（4-5 月整理区）；第三批 100±3 元（控股股东受让价 + 整数关口）；极端批：缩量企稳或恐慌新低后右侧确认（表格：批次/触发条件/仓位比例/逻辑）

### 8.3 既有持仓处理
